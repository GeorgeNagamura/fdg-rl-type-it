Help

This is a tool that creates formal representations of the Representational Level for Functional Discourse Grammar (HENGEVELD; MACKENZIE, 2008). Currently, the tool allows representation for any Layer in the Representational Level. Simply start by choosing the desired layer at “select head” and clicking the “Add head” button.

Once a head is added, its representation will be displayed under the “Select a head” area. On the right side of the screen, the available options for this Layer will be shown. You can now select a Semantic Role for this head, or add an Operator (consult Hengeveld & Mackenzie (2008) to see which Operators are available for each Layer). Under these options there is also a “Reset Representation” button. Clicking this button will erase the current representation and cannot be undone.

After a head is inserted you can add another head and so on. The index of each Layer will reflect its order of addition (starting with “i”, then “j”, etc., when a repeated variable is inserted).

If you wish to add a modifier, first select the variable for which the modifier applies on the right side of the screen, then select the variable that specifies the modifier, then mark the checkbox “Add as modifier?” and finally click “Add head”. The modifier head will also be available on the right side for adding a new head to it.

This tool also allows for coordinated members to be added. To do so, select “Coordination Frame” on the “Select a head” menu, then click “Add head”. This frame is now available on the right side, to receive a new head. After inserting a head on the Coordination Frame, the newly created head will automatically be selected to receive its specifications. In order to add another member to the Coordination, select it by clicking on the round checkbox (radio checkbox) to the left of the Coordination variable (right side of the screen). After at least two members of the Coordination have been added, it is possible to mark it as an Alternative Coordination, by clicking the button “Alt” (right side of the Function selection).

Finally, any Layer can receive a Lexical head. To do so, first select “Lexeme” in the “Select a head” menu and a textbox will appear. Type in the box the desired lexeme and then click the “Add head” button.

When you are satisfied with the representation, click the “Copy to Clipboard” button, and paste it (using the edit command or “ctrl(also command)-v” in your text editor of choice.


This tool is a work in progress and has many limitations. First, it cannot recognize faulty representations, for example, it is possible in this program to create a Propositional Content with a Configurational Property as a head, even though it is not a valid representation in the Grammar.

Second, the order of indexes reflect the order by which they were added and not the linear order of succession in the representation. If the order of indices are wrong you will have to manually correct them (with your text editor of choice) or reset the representation.

Finally, there may be other specifications you want to add to the representation that are not contemplated here. In this case, e-mail me at genaga.trad@gmail.com and I will see if it is possible to add your desired feature to the program, or if you have any other type of feedback.
